Tarjeta Grafica RTX 3080 TI


La GPU GeForce RTX 3080 Ti de NVIDIA es una auténtica bestia. En el análisis que publicamos a principios del pasado mes de junio nos demostró que esta tarjeta gráfica es capaz de mover con soltura cualquier juego actual a 2160p y con la máxima calidad gráfica posible.


Procesador Intel i7


La CPU i7 es más adecuada para las aplicaciones especializadas, que consumen muchos recursos y están orientadas al rendimiento, así como para los juegos de gama alta. El i7 también es mejor para ejecutar varias tareas simultáneamente.


Case Gaming


Case para PC con 6 ventiladores ARGB, fundas para juegos de computadora con puerto tipo C y USB 3.0.
